HTML基础

网页基本结构

<!doctype html>
<html>
	<head>
		<meta charset="utf-8">
		<title>第一个网页</title>
	</head>
	<body>

	</body>
</html>

元素

  • 常用元素：html, body, div, span, iframe, h1, h2, h3, h4, h5, p, a, img, ol, ul, li, form, label, table, tbody, thead, tr, th, td, footer, header, br, hr, select, input, textarea, button

标签

  • 大多数元素书写的时候需要有开始和结束标签
  • img, br, hr 不需要结束标签
  • 建议所有标签都小写
  • 大多数标签可以嵌套

属性

  • 大多数标签可以有属性
  • 有些标签有自己特有的属性

CSS基础

选择器

  • 基础的选择器包括：类型选择器、id选择器、class选择器

注释语句

  • CSS 的注释语句以 /* 开始，以 */ 结束

样式代码的引用

  • 外部引用
  • 内联引用
<!doctype html>
<html>
	<head>
		<meta charset="utf-8">
		<title>第一个网页</title>
		<link rel="stylesheet" href="css/style.css">
		<style type="text/css">
			//这里写css代码
		</style>
	</head>
	<body>

	</body>
</html>

JS基础

js代码的引用

  • 引用外部文件
  • 在页面中放置js代码
<!doctype html>
<html>
	<head>
		<meta charset="utf-8">
		<title>第一个网页</title>
		<link rel="stylesheet" href="css/style.css">
	</head>
	<body>

		<script>
			//这里写js代码
		</script>
		<script src="js/index.js"></script>
	</body>
</html>
